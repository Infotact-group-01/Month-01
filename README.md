# Advanced Threat Intelligence Platform (TIP) & Dynamic Policy Enforcer

## Project Overview
The Advanced Threat Intelligence Platform (TIP) & Dynamic Policy Enforcer is a cybersecurity-focused project developed for the Finance and Banking sector. The main objective of this project is to build an automated system capable of collecting, processing, and managing cyber threat intelligence from multiple Open Source Intelligence (OSINT) feeds.

Traditional security systems are often unable to respond quickly to rapidly evolving cyber threats such as malware attacks, phishing campaigns, and malicious IP activities. This project aims to improve proactive defense mechanisms by automating threat intelligence aggregation and enabling dynamic security policy enforcement.

The platform is designed to gather threat indicators such as malicious IP addresses, domains, and suspicious URLs from public OSINT sources including AlienVault OTX and VirusTotal. The collected data is intended to be cleaned, normalized, and stored in a centralized MongoDB database for further analysis and monitoring.

In addition to threat aggregation, the project also focuses on automated network protection using Linux firewall technologies. The future implementation of the Dynamic Policy Enforcer module will allow the system to automatically update firewall rules and block malicious connections in real time.
